# Git Lab 2 – Branching

## Commands I learned
• git branch → create a new branch
• git checkout → switch to a branch
• git merge → combine changes from another branch
• git push → upload branch to GitHub
• git pull --allow-unrelated-histories → merge remote history with local

## My Understanding
Branching lets me work on features separately without breaking the main project.  
It feels like creating a parallel timeline, then merging it back when ready.
