# Git Lab 1 – Basics

## Commands I learned
- git init → start a repo
- git add → stage files
- git commit → save snapshot
- git status → check repo state
- git log → view history

## My Understanding
Git has 3 areas:
1. Working Directory → where I edit files
2. Staging Area → files prepared for commit
3. Repository → permanent history

Staging feels like packing items before storing them. Commit is like sealing the box and putting it in storage.
